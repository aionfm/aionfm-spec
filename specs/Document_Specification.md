# Document Specification

This specification describes all documents required to define, implement and deploy the **AionFM** temporal foundation model and its platform ecosystem. Each document has a clear purpose, intended audience, key sections and estimated length.  Dependencies indicate when a document relies on information defined in earlier documents.  Rust developers should be able to build the core AionFM model in the `aionfm` repository and create supporting platform repositories such as `aionfm-ui`, `aionfm-api` and `aionfm-sdk` based on these specifications.
